# PokeExplorer

PokeExplorer is a React application that consumes the PokéAPI.

## Features

- Search Pokémon by name
- Display Pokémon image, stats, type and abilities
- Dynamic UI updates based on fetched data
- Error handling with custom "Not Found" state
- Scrolling Pokémon name list (marquee style)

## Technologies Used

- React
- Vite
- JavaScript (ES6+)
- PokéAPI
- GitHub Pages (deployment)
